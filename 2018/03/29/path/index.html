参考文档：https://nodejs.org/dist/latest-v9.x/docs/api/path.html

path 是 Node 本身提供的一个核心模块，专门用来处理路径。

使用它的第一步就是先加载：

path.basename

获取一个路径的文件名部分

path.dirname

获取一个路径的目录部分

path.extname

获取一个路径的后缀名部分

path.parse

将一个路径转换为一个对象，得到路径的各个组成部分

path.format(pathObject)

将具有特定属性的对象转换为一个路径

path.join

将多个路径拼接为一个

path.isAbsolute

判断一个路径是否是绝对路径

path.normalize(path)
